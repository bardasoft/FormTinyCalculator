# FormTinyCalculator
Calculator written using Xamarin Forms (no XAML) in +-75 lines of code (if you remove the superfluous braces)
No third party libraries used, just what is available in Xamarin.

Clients:

- Android
- iOS
- Mac
- Windows Universal Platform
